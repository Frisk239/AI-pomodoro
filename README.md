### 选题：基于web全栈的智能番茄钟学习辅助系统
### 简介：这是一个功能丰富的全栈Web应用，结合番茄工作法、AI学习伙伴、虚拟自习室和数据分析，帮助用户提高学习效率。

### 一、功能介绍
### 1.番茄钟计时器
- 25分钟工作 + 5分钟短休息的经典番茄工作法
- 每4个番茄钟后自动进入15分钟长休息
- 可视化进度环显示剩余时间
- 任务名称记录和时长统计

### 2.AI学习伙伴
- 基于智谱GLM-4大模型的智能对话
- 学习计划制定和问题解答
- 快速操作按钮，一键获取常见学习建议
- 实时聊天界面，支持Markdown格式

### 3.虚拟自习室
- 实时多人聊天室功能
- 创建和加入自习房间
- 在线用户列表显示
- WebSocket实时通信

### 4.待办事项管理
- 添加、完成、删除待办事项
- 预估任务时长设置
- 一键启动番茄钟专注任务
- 已完成/待完成任务分类

### 5.数据统计分析
- 专注时长分布图表
- 时间趋势分析
- 专注时段热力图
- 个人学习数据统计
- 图表可视化分析

### 二、技术栈
### 1.前端技术
- index.html 主页面
- style.css 样式文件
- script.js 前端交互逻辑

### 2.后端技术
- app.js Express服务器
- database.js 数据库配置
- check-db.js 数据库检查
- .env 环境变量配置

### 3.数据库技术
- pomodoro.db SQLite数据库
- package.json 项目依赖配置
- README.md 项目说明文档

### 三、环境要求
- Node.js v18.0.0 或更高版本
- npm 6.0+
- SQLite 数据库
- Web 浏览器支持 HTML5、CSS3 和 JavaScript

### 四、安装步骤
### 1. 克隆项目到本地
- 可从gitcode上下载项目
- https://gitcode.com/weixin_70454796/CAI

### 2.安装依赖
```javascript
	npm install
```

### 3.配置环境变量
- 创建 .env 文件，并添加以下内容：
```javascript
	ZHIPU_API_KEY=your_zhipu_api_key_here
	PORT=3000
```

### 4.启动服务器
```javascript
	npm run start
```

### 5.访问应用
- 打开浏览器，访问 http://localhost:3000
- 开始使用番茄钟学习辅助系统
